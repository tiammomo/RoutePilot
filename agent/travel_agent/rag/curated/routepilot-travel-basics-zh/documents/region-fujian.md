# 福建旅行区域与行程组织

## 先选择沿海或山地主题

福建可按福州及闽都文化、泉州及海丝文化、厦门城市与滨海、漳州与土楼片区、武夷山及闽北生态茶文化，以及三明泰宁等山水方向拆分。城市与山地目的地分布较散，短行程应围绕一个门户和一条主线，不宜从厦门、泉州一路跨到武夷山再当天折返。

## 行程建议

- 厦门—泉州适合沿海城市联游，但应给泉州历史街区和场馆独立时间。
- 土楼不是一个单点名称；出发前必须确认具体楼群、入口、接驳和返程方式。
- 武夷山适合单独安排住宿与天气缓冲，不和沿海城市高密度拼接。
- 福州可作为闽都文化主线，也可按交通条件与平潭等方向组合，但海岛交通需另行核验。

## 适合谁

偏历史城市和非遗可选泉州—厦门或福州；偏自然与茶文化可选闽北；家庭旅行宜减少同日长途公路加高强度步行。自驾进入山地或村落时，要确认停车和道路条件。

## 必须实时核验

台风、强降雨、海上交通、景区与土楼开放、预约、山区道路、演出和茶事活动必须实时查询。本指南不保证轮渡、海岛或漂流项目运营。
