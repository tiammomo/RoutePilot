# 广西旅行区域与行程组织

## 把山水、滨海与边关分开规划

广西可按桂林—阳朔等桂北山水，南宁城市与壮乡文化，北海—钦州—防城港的北部湾滨海，以及崇左—凭祥等边关文化方向组织；柳州、河池、百色和贺州也各有独立主题。桂林、南宁和北海虽然可串联，但短行程中全部加入会造成大量转场。

## 行程建议

- 桂林与阳朔应按到达站、住宿和山水体验拆分，不要只按“同一目的地”估时。
- 南宁可作为城市基地或前往边关、北部湾的中转，但不同方向不宜同日往返。
- 北海及海岛行程受海况影响明显，必须保留陆上备选。
- 边境地区自驾、口岸或跨境活动要确认当日开放、证件与管制，不使用旧攻略推断。

## 适合谁

首次山水旅行可聚焦桂北；希望兼顾城市、美食与民族文化可选南宁及一个邻近方向；滨海度假宜减少换酒店。带老人儿童时，避免把溶洞、竹筏、山地和长途公路连续安排在一天。

## 必须实时核验

江河水位、竹筏和游船、海况与轮渡、台风暴雨、边境与口岸政策、山区道路和景区开放必须实时查询。知识库不承诺跨境资格、漂流安全或固定交通时间。
