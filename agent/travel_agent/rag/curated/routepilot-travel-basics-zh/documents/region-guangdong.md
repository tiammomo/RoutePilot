# 广东旅行区域与行程组织

## 先拆成城市群

广东不是单一城市目的地。常见旅行结构可分为广州—佛山的岭南城市文化、深圳的现代城市与滨海休闲、潮州—汕头—揭阳的潮汕文化与饮食，以及粤北、粤西等自然和地方文化线路。大湾区交通联系较强，但“可以到达”不等于适合每天跨城往返。

## 行程建议

- 首游三四天宜选择一个主城市群，不要同时覆盖广府、客家、潮汕和滨海全部主题。
- 以美食为主时应按街区和用餐时段安排，避免在同一餐期跨城追店。
- 广州与佛山可按实际住宿和返程方向组合；深圳与珠海等跨口岸或跨水域安排要另留手续与候车缓冲。
- 潮汕地区应围绕一至两个住宿基地组织，先确认城际到达点和市内接驳。

## 适合谁

城市文化和美食首游适合广州—佛山；偏现代城市体验可选深圳；对非遗、古城和地方饮食兴趣明确时可独立安排潮汕。家庭行程应降低连续跨城频率，炎热或强对流天气下给室内备选。

## 必须实时核验

口岸政策、跨境要求、轮渡和城际班次、台风与强降雨预警、景区预约、餐厅营业和道路状态必须实时查询。知识库不对具体餐厅、班次或通关时间作保证。
