# 云南旅行区域与行程组织

## 云南需要按方向成线

云南常见结构包括昆明及滇中、昆明—大理—丽江—香格里拉的滇西北、腾冲—保山—德宏方向的滇西、普洱—西双版纳的滇南，以及建水—蒙自等滇东南人文线路。各方向之间跨度大、气候与海拔差异明显，第一次旅行不应同时追求“把云南主要城市都走完”。

## 行程建议

- 昆明可作为到达缓冲和换线节点，不必机械安排大量打卡点。
- 滇西北线路应逐步提高海拔和活动强度，给天气、交通和身体状态留余量。
- 滇南主题偏热带生态与地方文化，通常不适合与滇西北在短假期强行拼接。
- 边境地区、徒步和自驾线路必须先确认开放、证件、交通与救援条件。

## 选择逻辑

首次到访且偏人文山水，可选昆明加滇西北的一部分；偏自然生态和暖热气候可选滇南；偏历史、铁路和古城可看滇东南。带老人儿童时优先减少频繁换住宿和长距离公路移动，并慎重评估高海拔。

## 必须实时核验

天气、地质灾害预警、道路、航班和铁路班次、景区开放、边境管理、徒步线路与高海拔健康风险必须查询官方或 Provider。知识库不把网络“最佳季节”或固定车程当作承诺。
