# 北京旅行区域与行程组织

## 先按片区做选择

北京不适合把所有知名地点塞进同一天。首次到访可先在三类片区中取舍：中轴线与老城适合历史城市脉络；西北方向适合园林、校园周边与山水文化；长城等远郊目的地通常应独立成日。城市副中心、大运河和西山永定河等区域可以作为再次到访或主题旅行的扩展，而不是默认全部加入首游。

## 行程建议

- 只有两三天时，以一个核心住宿区为基点，每天围绕一个主片区活动。
- 参观型地点较多时，上午和下午各保留一个主目标，中间安排可取消的街区或公园。
- 远郊日不要再叠加需要预约、闭馆时间严格或步行量大的核心场馆。
- 带老人儿童时，优先减少地铁换乘与长距离步行；不要把“地图距离近”当作入口也近。

## 适合怎样组合

历史文化首游可从老城与中轴线展开；偏园林和博物馆可把西北片区单独成日；亲子家庭宜减少连续室内讲解点，并预留用餐和休息。跨城联游时，北京本身已需要较多交通和安检缓冲，不建议在短行程中频繁更换住宿。

## 必须实时核验

预约名额、开放日期、入园入口、临时展览、长城交通、极端天气和道路状态必须通过 Provider Gateway 或对应官方页面确认。本指南不保证任一场所当日开放，也不提供固定通勤时间。
