# 四川旅行区域与行程组织

## 先区分盆地城市与山地线路

四川幅员和海拔差异大。成都及周边适合城市生活、博物馆和都江堰—青城山等文化山水；乐山—峨眉山适合独立主题；川西高原、川北和藏羌文化线路需要更多交通、天气与身体适应准备；川南可围绕历史城市、美食和自然景观另成一线。不要把“成都出发能到”误解为适合当日轻松往返。

## 行程建议

- 首游短行程以成都为基地，加一个明确的周边方向。
- 乐山与峨眉山是否连排取决于步行能力、预约、住宿和返程，不默认一天完成。
- 川西线路应减少每日目标，设置机动日，并避免到达当天安排高强度活动。
- 自驾或包车不能替代道路、天气、海拔和司机资质核验。

## 适合谁

偏城市、美食和文化的旅客可优先成都及盆地周边；偏山水与户外者需要明确体能、季节和风险承受；带老人儿童时，优先选择医疗和交通保障更稳定的基地，谨慎增加高海拔线路。

## 必须实时核验

山区道路、地质与气象预警、景区预约、索道状态、海拔适应、森林防火和交通管制必须以官方实时信息为准。高原身体风险不能由旅行 Agent 作医疗判断。
