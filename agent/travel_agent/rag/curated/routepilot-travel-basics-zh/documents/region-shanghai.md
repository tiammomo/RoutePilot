# 上海旅行区域与行程组织

## 先判断旅行主题

上海适合按城市空间和生活方式组织，而不是按景点数量组织。首次到访可把外滩—人民广场—老城作为历史城市核心，把衡复地区等街区作为慢行体验，把浦东作为现代城市视角；杨浦滨江等工业遗产区域适合已有明确兴趣的人。黄浦江两岸看似相邻，但过江、进出站和步行入口会改变实际动线。

## 行程建议

- 一天只设一个主街区或一段滨水空间，减少东西两岸来回切换。
- 博物馆、展览和演出应先锁定时间，再把附近街区作为弹性内容。
- 亲子或雨天行程可以室内场馆为主，但仍应核验预约和临时闭馆。
- 夜景不是必须项；若同行人容易疲劳，可把城市夜间体验放在住宿地附近。

## 住宿与联游

选择住宿时优先看首日抵达、末日离开和主要片区的公共交通关系，而不是只看行政区名称。江南联游可从上海衔接江苏或浙江，但短途旅行中每天换城会明显压缩有效游览时间。若重点是街区、美食和展览，留在同一城市深度体验通常比增加城市数量更合适。

## 必须实时核验

展览档期、场馆预约、轮渡和游船班次、演出库存、商业营业、道路管制与天气必须实时查询。本指南不把历史规划文件中的项目状态当作当前营业事实。
