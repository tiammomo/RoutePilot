# 老人、儿童与无障碍旅行规划

多人旅行不能只用“成人数量”代表行动能力。规划前应确认是否有人需要减少步行、避免楼梯、定时休息、携带婴儿车或轮椅，以及是否存在饮食、用药和感官环境需求。只收集会影响计划的信息，并避免在知识库或公开证据中存储不必要的健康隐私。

带老人或儿童时，应减少连续站立、频繁换乘和没有退路的长距离活动。每天保留明确的休息点、室内替代项和提前结束方案。用餐与卫生间不是行程之外的附属信息；它们会直接影响活动持续时间和路线可行性。

无障碍路线需要端到端验证。场所存在无障碍入口，不代表从车站到入口、场内主要区域、卫生间和返程路线全部可用。电梯检修、坡度、路面、接驳车辆和预约要求都可能改变结果。RoutePilot 应分别记录每个环节的来源和更新时间，并把尚未确认的环节标为风险。

交通选择要同时考虑换乘次数、步行距离、候车环境、是否容易找到座位以及发生延误后的替代路线。网约车或出租车可能减少步行，但车型、儿童座椅、轮椅容纳和上车点仍需实时确认，不能默认可用。

本知识只提供规划方法，不构成医疗、安全或无障碍设施保证。出发前应通过交通运营方、场所官方渠道或 Provider Gateway 核验设施状态、预约规则和临时变化；存在重要健康或照护需求时，应由旅行者结合专业建议作出决定。
