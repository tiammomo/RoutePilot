# RoutePilot 中文旅行知识包

这个知识包包含 RoutePilot 团队编写并复核的稳定旅行决策方法，以及北京、上海、广东、浙江、江苏、四川、云南、福建、海南、陕西、广西、湖南首批 12 个省级地区的旅行结构知识。地区内容回答“去哪些片区、怎样组合、什么必须实时核验”，不包含营业时间、票价、班次、天气、库存、签证政策或目的地安全公告等易变事实。

`manifest.json` 是发布清单。每篇正文都绑定 SHA-256、来源版本、MIT 许可、负责人、复核时间和固定检索问题。地区指南还必须声明 `upstream_sources`，记录审核时使用的官方页面、发布方、版本和观察时间；上游页面仅用于来源追溯，不会被整页抓取或按 MIT 重新许可。

修改正文时必须同时提升 `source_version`、更新 `content_sha256`、`review` 和 `corpus_revision`，否则校验器会拒绝发布。具体地点与实时事实仍由 Provider Gateway 提供。
