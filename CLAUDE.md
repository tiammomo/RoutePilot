# ShuaiTravelAgent Claude Guide

## Project Overview

ShuaiTravelAgent is an AI travel assistant project with:

- Frontend (`frontend`): Next.js app
- API (`web`): FastAPI service
- Agent (`agent`): LangGraph-based reasoning and tool execution

## Runtime Ports

- Frontend: `http://localhost:33001`
- API: `http://localhost:38000`

## Current API Route Modules

```text
web/src/routes/
- api_docs.py
- chat.py
- city.py
- errors.py
- health.py
- model.py
- session.py
```
